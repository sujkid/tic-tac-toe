
# TIC-TAC-TOE

An SPA for the Tic Tac Toe game built in Javascript.
My first game project with General Aasembly Boston's WDI.

## Technologies used

- HTML
- CSS
- Javascript
- jQuery
- Bootstrap
- Git/Github
- Webpack

## User Stories

As a user,

- I can sign up.
- I can sign in.
- I can change password.
- I can sign out.
- I can create new games.
- I can view how many games I have played.
